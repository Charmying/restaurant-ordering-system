# Frontend (client applications)

**English** | [繁體中文](./README.zh-TW.md)

This folder hosts customer-facing and staff-facing web clients. The active implementation is **`angular-app/`** (Angular). Additional apps (for example React) can be added as siblings later.

## Quick start (Angular)

```bash
cd angular-app
npm install
npm start
```

The app runs at `http://localhost:4200/` by default. See [`docs/SETUP.md`](./docs/SETUP.md) for details.
