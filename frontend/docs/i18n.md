# i18n Setup Guide

## Quick Start

### In Components

```typescript
import { TranslateModule } from '@ngx-translate/core';

@Component({
  imports: [TranslateModule],
  template: `<h1>{{ 'layout.header.title' | translate }}</h1>`
})
export class MyComponent {}
```

### Switch Language

```typescript
import { TranslateService } from '@ngx-translate/core';

export class MyComponent {
  private translate = inject(TranslateService);

  switchLanguage(lang: string) {
    this.translate.use(lang); // 'en' | 'zh'
  }
}
```


---

## Adding New Keys

1. Add to **both** `en.json` and `zh.json`
2. Follow naming convention in `docs/architecture/i18n-naming.md`
3. Use structural keys, not text-based keys

---

## Configuration

See `src/app/app.config.ts` for TranslateModule setup.

Default language: `zh`
